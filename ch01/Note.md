#### matplotlib

- 그래프 그리기

```
import matplotlib.pyplot as plt

...
plt.plot (x, y)
plt.xlabel("x") - x 축 이름
plt.ylabel("y") - y 축 이름
plt.title('') - 그래프 제목
plt.legend() - 그래프 설명 (우측 상단 모서리에)
plt.imshow() - 이미지 표시
plt.show()
```
